Audit repeated agent rules before compression gets risky.

RuleMeter is a report-only CLI for checking duplicated instruction text, alias break-even, token cost, cache-prefix suitability, and high-risk rules that should stay explicit.

Report first, rewrite never.

RuleMeter does not edit files. It gives maintainers evidence for what is duplicated, what can be removed, and what should remain explicit because it protects identity, approvals, tests, logs, or security.

  1. 01 Discover instruction files

    Use presets for Codex, Claude, Copilot, Antigravity, or all known agent surfaces.

  2. 02 Measure token economics

    Compare raw text, alias cost, legend cost, break-even, and duplicate deletion savings.

  3. 03 Keep risky rules explicit

    Flag sensitive instruction families so safety-critical rules are not compressed away.

What the report separates

Exact duplicates

Shows when deleting repeated text beats introducing an alias.

High-risk rules

Keeps identity, PII, approval, test, log, and security rules visible.

Similar candidates

Optional lexical-overlap review prompts, never automatic semantic dedupe.

CI gates

Fail on duplicates, risks, candidates, or experimental similar-rule findings.

Use it before publishing pressure.
